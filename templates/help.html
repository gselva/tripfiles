Registration, Login and Logout
To begin using TripFiles, use the on the top right of the web page to begin the registration. During registration, you'll need a google login (which you can get for free here) and your payment details.

To login into TripFiles, use the button on the top right of the web page. You should use the same google login your registered with.

To Logout of TripFiles, use the button on the top right of the web page.
Account Management and Subscription
Once logged-in into TripFiles, you can manage your Account using the button on the top right of the web page. You can update the details that go into your Mileage report on the Account page.

On the same page, you can also manage your TripFiles Subscription. TripFiles uses (Spreedly.com) to manage your subscriptions. You'll be taken there during Subscription update. You can return to TripFiles from Spreedly using the TripFiles link on Spreedly page (displayed at the bottom of Spreedly page).
Creating and updating Mileage expenses
Use the Home menu link (visible beneath TripFiles Logo on top left) to go to New Trip page. The left panel on this page provides various fields that you can optionally fill to create a trip. One trip corresponds to one row in your mileage report. You can create as many trips as needed. There can be multiple trips on the same date (Tip: You can save a trip and then simply re-use the mileage values by changing just the date).

From the Report page (see below), you can use the Update button on any row of the Mileage report to update that particular trip. The Update Trip page is similar to New Trip page. Do note that when you update Trip Date on an update page it will update the date rather than create a new trip (unlike the Tip provided earlier for creating New Trip).
Viewing and Downloading Mileage expense reports
Use the Reports menu link to go to Reports page. Select the date range for the report to view the report. You can download a PDF copy of the report using the download button (shown only if the report can be downloaded).

Note: The PDF report is formatted according to the specific template used within your organization. Please contact customer support if this has changed.
